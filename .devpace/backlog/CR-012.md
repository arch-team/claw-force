## MoS 指标定义

### 核心指标

| 指标 | 定义 | 目标值 | 度量方式 |
|------|------|--------|---------|
| **部署成功率** | `cdk deploy` 一次成功（无需手动修复）的比率 | >= 90% | deploy 日志 |
| **对话响应率** | AI 员工成功返回回复的比率（vs 超时/错误） | >= 95% | OpenClaw 日志 |
| **首次响应延迟** | 从用户发送消息到 AI 员工首次回复的时间 | < 10s | 飞书消息时间戳差 |
| **飞书连接稳定性** | 飞书 WebSocket 连接持续在线时间占比 | >= 99% | OpenClaw channel status |
| **端到端恢复时间** | `cdk destroy + deploy` 后全功能恢复所需时间 | < 15min | 部署日志 |

### 验证检查清单

部署后执行以下端到端验证：
- [ ] `cdk deploy` 成功完成
- [ ] OpenClaw Control UI 可通过 ALB URL 访问
- [ ] 飞书 Bot 在线（`openclaw channels status` 显示 feishu connected）
- [ ] 通过飞书发送消息，AI 员工正确回复
- [ ] 通过 Hooks API（`POST /hooks/agent`）发送消息，AI 员工正确回复
- [ ] `cdk destroy + cdk deploy` 后所有功能自动恢复
