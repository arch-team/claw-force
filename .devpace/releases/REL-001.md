## Changelog

### Features

- [CR-001] CDK TypeScript 基础设施自动化部署（PF-001）- 将 PoC 手动部署转为 CDK TypeScript，涵盖 EC2+SG+IAM+EBS 全栈，内嵌 6 条 PoC 修复项
- [CR-002] Bedrock 模型验证与配置增强（PF-002）- Inference Profile 格式适配、模型发现脚本、IAM 策略自动配置
- [CR-003] ALB+HTTPS+WAF 企业级网络安全基线（PF-003）- ALB 双模式架构（ALB vs 直连），WAF WebACL 速率限制
- [CR-004] CloudWatch 基础运维可观测性（PF-004）- 日志组、CPU/StatusCheck 告警、CloudWatch Agent 集成
- [CR-005] CDK 基础设施自动化测试覆盖（PF-009）- 39 个 Jest 测试覆盖所有 construct 和 stack 双模式

## Release Notes

### BR-001: 企业级 AI 员工平台基础设施

- **IaC 自动化部署**：一键 `cdk deploy` 创建完整 AWS 基础设施（EC2 + IAM + SG + EBS + CloudWatch），支持参数化配置
- **Bedrock 模型适配**：自动使用 Inference Profile 格式，支持 Claude Haiku 4.5 / Sonnet 4.5 / Opus 4.5
- **企业级安全基线**：ALB + WAF 双模式（支持 ALB 安全层或直连快速验证），Security Group 最小化
- **运维可观测性**：CloudWatch 日志自动采集、CPU 和状态检查告警、CloudWatch Agent 集成
- **自动化测试**：39 个测试用例覆盖所有基础设施组件，确保 IaC 变更安全
